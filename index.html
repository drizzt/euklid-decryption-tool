EUKLID DECRYPTION TOOL

Client Side

Description:

The signal ledgers are encrypted using Rijndael with 128 bit key size and block size of 256 bit (Block sizes of 128, 160, 192, 224, and 256 bits are supported by the Rijndael algorithm, but only the 128-bit block size is specified in the AES standard).

The decryption is done client side by using the js-mcrypt library.
